# Exoplanet Data Analysis 🪐

A Python-based statistical analysis of confirmed exoplanet data, exploring
planetary properties and their distributions using real astronomical datasets.

## 📌 Overview

This project analyses a dataset of confirmed exoplanets to uncover trends
in planetary mass, orbital period, radius, and discovery methods. It uses
core data science libraries to clean, process, and visualise the data.

## 📊 What This Project Does

- Loads and cleans exoplanet CSV data using Pandas
- Explores distributions of mass, radius, and orbital period
- Visualises correlations between planetary properties
- Identifies trends across different discovery methods

## 🛠️ Libraries Used

- `pandas` — data loading and manipulation
- `matplotlib` — plotting and visualisation
- `numpy` — numerical operations
- `scipy` — statistical analysis

## ▶️ How to Run

1. Clone the repository or download the files
2. Install dependencies:
   pip install pandas matplotlib numpy scipy
3. Run the script:
   python exoplanet_analysis.py

## 📈 Sample Output

The script generates plots including:
- Distribution of orbital periods
- Mass vs Radius scatter plot
- Discovery method breakdown
